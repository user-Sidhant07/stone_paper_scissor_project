# stone_paper_scissor_project
**A simple stone paper scissor game built using python**

##🚀 Features
*Player vs Player*
*Player vs Computer*
*Random choice for player*
*Random computer choices*
Input validation
Replay option
##🛠️ Tech Used
Python
Random module
##▶️ How to Run
Clone the repository
Run the file: python stone_paper_scissor.py
